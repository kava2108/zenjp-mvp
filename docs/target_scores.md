# 期待スコア設定

## 目標

3銘柄が以下の範囲内に収まるように定数を調整する。

## 期待スコア

### 7203 トヨタ自動車
- **総合スコア:** 70-80点（B+ランク）
- **Valueスコア:** 75-85点（配当重視企業として高評価）
- **Growthスコア:** 60-70点（安定成長）
- **Momentumスコア:** 70-80点（安定需給）

**理由:**
- 安定配当企業として知られる
- 配当利回りが高い（8.68%）
- 成長率は安定的だが爆発的ではない
- 日本を代表する優良企業

### 6758 ソニーグループ
- **総合スコア:** 75-85点（Aランク目標）
- **Valueスコア:** 65-75点（成長企業のためPER高め許容）
- **Growthスコア:** 80-90点（高成長企業）
- **Momentumスコア:** 75-85点（活発な需給）

**理由:**
- エンタメ・ゲーム・半導体で高成長
- グローバル企業として評価高い
- イノベーション企業として市場から期待される
- 配当利回りは低いが成長性で評価

### 9984 ソフトバンクグループ
- **総合スコア:** 55-70点（Bランク）
- **Valueスコア:** 50-60点（投資会社のため指標に特殊性）
- **Growthスコア:** 70-80点（投資先成長を反映）
- **Momentumスコア:** 変動大（50-80点）

**理由:**
- 投資会社のためPER/PBRの解釈が特殊
- 成長率は投資先次第で変動
- ボラティリティが高い
- 配当利回りは低め

## スコアリングの基本方針

1. **バリュー企業（トヨタ）**: 配当・安定性を重視
2. **グロース企業（ソニー）**: 成長性・将来性を重視
3. **特殊企業（SBG）**: バランス型、ボラティリティ許容

## 調整の優先順位

1. **最優先**: ソニーのスコアを30点引き上げ（48→75-85点）
2. **優先**: SBGのスコアを20点引き上げ（34→55-70点）
3. **微調整**: トヨタを2-10点引き上げ（69→70-80点）
