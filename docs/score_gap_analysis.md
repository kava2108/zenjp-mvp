## 調整が必要な定数

### 🔥 緊急（すべての銘柄に影響）

1. **GROWTH_RATE_FIXED**: 10.0 → 削除または銘柄別設定
   - 現状: すべて50点固定
   - 問題: 成長企業も安定企業も同じスコア
   - 対策: 銘柄ごとに成長率を設定
     - トヨタ: 5% → 60-65点
     - ソニー: 15% → 85-90点
     - SBG: 12% → 75-80点

2. **RSI_IDEAL_MAX**: 70.0 → 75.0
   - 現状: RSI 70超で急激にスコア低下
   - 問題: ソニー（70.65）、SBG（75.35）が不当に低評価
   - 対策: 理想範囲を40-75に拡大

3. **RSI_OVERBOUGHT**: 70.0 → 80.0
   - 現状: 70超で買われすぎ判定
   - 問題: 70-75のRSIは通常の上昇トレンド
   - 対策: 80超を買われすぎとする

### ⚠️ 優先度高

4. **VOLUME_INCREASE_MULTIPLIER**: 10.0 → 5.0
   - 現状: 出来高変化率の影響が大きすぎる
   - 問題: わずかなマイナスで0点
   - 対策: 係数を下げて影響を緩和

5. **GROWTH_BASELINE**: 10.0 → 5.0
   - 現状: 成長率10%で50点
   - 問題: 安定成長企業（5-10%）が低評価
   - 対策: 基準を5%に下げる

6. **GROWTH_PERFECT_THRESHOLD**: 20.0 → 15.0
   - 現状: 20%以上で100点
   - 問題: 現実的には15%でも高成長
   - 対策: 閾値を下げる

### 📌 優先度中

7. **PER計算の確認**
   - トヨタのPER 1.15倍は異常
   - 株価 / EPS の計算が正しいか確認
   - データが正しければ計算ロジックの問題

8. **DIVIDEND_MULTIPLIER**: 20.0 → 15.0
   - 配当スコアの影響をやや抑える
   - 成長企業が不利にならないように

## 調整優先順位

### フェーズ1: 緊急修正（30分）
1. RSI関連定数を調整（RSI_IDEAL_MAX, RSI_OVERBOUGHT）
2. Growthスコアを銘柄別に設定
3. 再計算して効果確認

### フェーズ2: 詳細調整（60分）
4. VOLUME_INCREASE_MULTIPLIERを調整
5. GROWTH_BASELINE / GROWTH_PERFECT_THRESHOLDを調整
6. PER計算ロジックを確認・修正
7. 再計算して期待値との比較

### フェーズ3: 微調整（30分）
8. 期待値に達していない項目を個別調整
9. 最終確認
