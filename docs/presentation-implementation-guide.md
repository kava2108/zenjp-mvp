## 🚀 クイックスタート（15分で完成させるコツ）

### ステップ1: テンプレート選択（2分）

**PowerPoint:**
```
1. PowerPoint 起動
2. 新規プレゼンテーション → 「デザインテンプレート」
3. 推奨: 「Dividend」または「Ion」（洗練、ビジネス向け）
4. カラースキーム変更: Blue to Navy/Light Blue
```

**Keynote（Mac）:**
```
1. Keynote 起動
2. テンプレート選択 → 「Blank」
3. フォント: Helvetica Neue / 12-56pt
```

**Google Slides:**
```
1. Google Drive → 新規 → Google Slides
2. テンプレートギャラリー → 「Modern Writer」
3. カラー編集: Blue系に統一
```

### ステップ2: 基本設定（3分）

すべてのスライドに共通で適用：

```
【マスター設定】
- スライドサイズ: 16:9 ワイドスクリーン
- フッター: 「ZenJP MVP | Seed Round」
- ページ番号: 右下に表示
- フォントセット: 見出し Montserrat / 本文 Inter
```

### ステップ3: 各スライド作成（10分）

以下の順序で、copy-paste + 編集：

1. スライド1-10の基本テキストをコピー
2. フォント＆色を統一
3. 画像＆グラフを挿入
4. アニメーション設定

## 🎨 高度なデザイン技法

### グラデーション効果

```
【スライド1の背景】
- スタート: #1F2937 (濃紺)
- エンド: #6366F1 (薄紫)
- 角度: 45度（左上→右下）
- PowerPoint: 「デザイン」→「塗りつぶし」→「グラデーション」
```

### アニメーション設定

```
【各スライド共通】
- スライド切り替え: フェードイン（0.3秒）
- PowerPoint: 「切り替え」→「フェード」→「0.30秒」

【テキストアニメーション】
- スライド2（問題提起）: テキストを1行ずつ出現
  - PowerPoint: 「アニメーション」→「ワイプ」→「順序通り」
  
- スライド7（パフォーマンス）: 数値をカウントアップ
  - PowerPoint: 「アニメーション」→「出現」→「カスタマイズ」
  - 時間: 1秒

【グラフアニメーション】
- スライド8（ロードマップ）: マイルストーン点灯
  - PowerPoint: 各マーカーに「光」エフェクト
  - 間隔: 0.5秒
```

### マスタースライド設定

```
【全スライド統一】
1. PowerPoint: 「表示」→「マスター」→「スライドマスター」
2. 各スライドレイアウトを編集
3. 以下を追加:
   - フッター: 左「ZenJP MVP」/ 右「2026年2月」
   - ロゴ: 右上隅に小さく（60×60px）
   - ページ番号: 右下（12pt）
4. 「閉じる」で保存
```

## 📸 スクリーンショット・画像ガイド

### 必要な画像（4枚）

#### 1. ZenJP フロントエンド（スライド4用）
```
取得方法:
1. ブラウザ: http://localhost:3000
2. F11で全画面表示
3. Cmd+Shift+4（Mac）or Win+Shift+S（Windows）
4. スクリーンショット撮影
5. PNG形式で保存: screenshots/frontend-1.png

仕様: 1920×1080px, 96dpi以上
```

#### 2. API応答タイムグラフ（スライド7用）
```
取得方法:
1. API ドキュメント: http://localhost:8000/docs
2. /api/score/{stock_id} のレスポンス時間を記録
3. Excel/Google Sheets で可視化
4. PNG形式で保存: screenshots/performance-graph.png

グラフ仕様: 棒グラフ, 平均値ライン表示
```

#### 3. ロードマップチャート（スライド8用）
```
取得方法: 
1. PowerPoint 内で直接作成（図形＋テキスト）
2. または Lucidchart / draw.io で作成
3. PNG/SVG 形式で保存: screenshots/roadmap.png
```

#### 4. ビジネスモデル図（スライド9用）
```
取得方法:
1. Excel で市場規模データをグラフ化
2. または Figma で図形作成
3. PNG形式で保存: screenshots/business-model.png
```

### 画像埋め込みのコツ

```
【PowerPoint内での挿入】
1. スライド選択
2. 「挿入」→「画像」→「ファイルから」
3. 画像ファイルを選択
4. リサイズ: コーナーのハンドルをドラッグ
5. 位置調整: 上下左右の配置ボタンで中央配置
6. 複数画像: グループ化して整列（「配置」→「配置」）
```

## 🎯 デモ当日の使用方法

### プレゼン前夜

```
1. ファイルをUSBとCloud（Dropbox/OneDrive）に保存
2. 表示マシンで動作確認
   - PowerPoint開く
   - 「スライドショー」→「最初から」でテスト再生
   - アニメーション動作確認
3. PDF版も準備（WiFi接続失敗時の代替）
```

### プレゼン当日

```
【セットアップ】
1. プロジェクター接続確認（HDMI / AirPlay）
2. PowerPoint立ち上げ
3. 「スライドショー」→「プレゼンターツール」
   - メインスクリーン: フルスライド表示
   - ノートPC: スライド+次スライド+Speaker Notes表示
4. タイマー設定（10分）

【実演中】
- Speaker Notes を見ながら話す
- ページアップ/アップキーで前後スライド移動
- Q：キーでレーザーポインター起動（カスタマイズ可）

【トラブル時】
- スライド飛ばし: 数字キー（例: 5キー → スライド5に移動）
- スライド非表示: Bキー（黒）or Wキー（白）
```
